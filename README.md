# Magnolia (Freelance Web Project)

A custom-built, multi-page website developed for a freelance client, focusing on clean architecture and smooth user transitions.

## 🛠️ Technologies Used
*   **Frontend:** HTML5, CSS3 (Custom styling)
*   **Logic:** JavaScript (Cart functionality, Search features, Order processing)
*   **Organization:** Semantic HTML for improved SEO and accessibility.

## 🔑 Key Features
*   **Search Functionality:** Custom `search.js` to handle client-side product filtering.
*   **Order System:** Integrated `order.js` and `cart.js` for a functional shopping experience.
*   **Responsive Design:** Optimized for various screen sizes to ensure high mobile performance.

## 📈 SEO Relevance
This project demonstrates my ability to write **clean, semantic code** which is the foundation of Technical SEO. By managing the site's structure manually, I ensured optimal crawling and indexing potential.
